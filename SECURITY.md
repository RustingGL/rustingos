# Security Policy

I will only update the newest version. I am not going to be updating previous versions.

## Supported Versions

| Version             | Supported               |
| :------------------ | :---------------------: |
| 0.0.0 - 0.8.9-alpha |           :x:           |
| 0.8.10-alpha        | :ballot_box_with_check: |

## Reporting a Vulnerability

Please put all vulnerabilities in the Issues section as a sub issue of 'Vulnerability Issues'.
